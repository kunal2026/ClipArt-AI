# 📱 React Native App

I built this project using React Native with the `@react-native-community/cli`. This README explains how I set up the project and how to run it locally.

---

## 🚀 Getting Started

Before running the project, I ensured my development environment was properly configured by following the official React Native setup guide (up to the “Creating a new application” step).

---

## ▶️ Step 1: Start the Metro Server

Metro is the JavaScript bundler used by React Native. I start it first.

From the root of the project, run:

```bash
npm start
```

or

```bash
yarn start
```

---

## 📲 Step 2: Run the App

With Metro running in one terminal, I open another terminal and run the app.

### For Android:
```bash
npm run android
```
or
```bash
yarn android
```

### For iOS:
```bash
npm run ios
```
or
```bash
yarn ios
```

If everything is set up correctly, the app should launch in the Android Emulator or iOS Simulator.

---

## ✏️ Step 3: Modify the App

Once the app is running, I edit:

```bash
App.tsx
```

To see changes instantly:

- **Android**: Press `R` twice or open Developer Menu (`Ctrl + M` / `Cmd + M`) and reload  
- **iOS**: Press `Cmd + R` in the simulator
